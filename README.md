# IoTBay

## Setup Database 
1. `Service` > `Database` > `Create Database`
2. `database = iotdb`
   `dbuser = iotueser`
   `password = admin`
   `URL = jdbc:derby://localhost:1527/iotdb`
3.  Click 'Run File' of `.sql` in `Web Pages` > `db`
